### Вариант 29

Библиотеки необходимые для сборки проекта:

* glib-2.0
* ncurses

Опционально (для отображения графика):

* gnuplot

Сборка проекта:

```bash
make
```

Запуск:

```bash
./bin/trapezoid
```
